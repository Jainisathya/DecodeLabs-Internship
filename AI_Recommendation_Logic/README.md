# 🤖 AI Smart Recommendation Hub

An AI-powered recommendation system built using Python and Streamlit.

## Features

- 🎬 Movie Recommendations
- 📚 Book Recommendations
- 🎵 Music Recommendations
- 🍔 Food Recommendations
- 🏋️ Fitness Suggestions
- 💻 Course Recommendations
- 👨‍🍳 Recipe Suggestions
- 💼 Career Recommendations
- ❤️ Favorites
- 🕒 Search History
- 📊 Analytics Dashboard
- 🌙 Light & Dark Themes

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- SQLite
- TF-IDF Vectorizer
- Cosine Similarity

## Installation

```bash
git clone <repository-url>
cd AI-Smart-Recommendation-Hub
pip install -r requirements.txt
streamlit run app.py
```
